2 关于reactnative

React Native 是有 Facebook 在15年开源的，基于 React 的跨平台框架。开发者可以使用 JS 和 React 来开发跨平台应用。

React 则是 FB 的一个前端框架，在13年开源。

2.1 简介

React Native 刚开始提出来只支持 iOS，应用了Visual Dom 和 Runtime 对 iOS 底层进行包装，所以使用的时候得学习 React Native 暴露在最顶层的接口，2016年开始 Android 版本逐渐完善。

目前 RN 只支持 iOS7 和 Android4.1 以上。

功能、交互效果简单，不需要原生系统支持，可考虑。较复杂也可以考虑 RN + 原生
