发表于 2017-02-12

需要安装的环境有：HomeBrew、Node、Hexo 以及运行博客的其他 npm 包。

申请域名

申请域名的过程还是比较简单的，可以通过国外的 Goddy 或者国内的阿里云购买。

我是从阿里云购买的 cn 域名，拍下身份证实名验证下即可，官网说需要3-5个工作日，实际上很快，我一天内就搞定了。

完成购买并且实名验证，当域名状态显示为 正常 的时候即可以使用了，登录阿里云账号 -> 管理控制台 -> 域名与网站 -> 云解析，如下图所示：

进入解析设置后，点击添加解析，解析方式有多种，我们使用的是 CNAME 方式，注意，第四个记录值填写的是 Github Page 的名字,其他跟下图一样即可，配置完点击保存。

保存后，会提示需要等待一段时间域名才能使用。

使用 CNAME 方式配置解析域名，需要再博客站点也配置 CNAME。在博客站点的 source 目录下新建一个 CNAME 的文件，其内容为我们的域名，如下图所示：

保存后，需要将我们的 CNAME 文件推送到 Github Page 中，再次使用 hexo d 命令即可。成功推送到 github 上我们的仓库会如下图所示：

建议每次使用 hexo d 命令前先使用 hexo clean 清理之前的静态页面

然后我们直接在浏览器输入我们的域名即可访问我们的博客了。
